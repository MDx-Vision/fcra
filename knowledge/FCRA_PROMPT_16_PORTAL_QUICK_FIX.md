# FCRA PROMPT 16: PORTAL QUICK FIX
## Fast PII & Inquiry Disputes via Online Portal

**Version:** 1.0
**Created:** January 13, 2026
**Use Case:** Quick fixes for funding/approval - NOT case building
**Method:** Online portal submission + phone follow-up
**Turnaround:** 3-7 days typical

## ‚ö†Ô∏è WHEN TO USE THIS PROMPT

**USE THIS (Quick Fix) When:**
- Client needs funding/loan approval ASAP
- Simple PII errors (typos, wrong DOB, old addresses)
- Random inquiries client wants removed
- Client just wants it FIXED, not suing
- Speed matters more than paper trail

**DON'T USE THIS When:**
- Building litigation case
- Need paper trail for escalation
- Complex violations (reinsertion, willful errors)
- Planning to send MOV/regulatory complaints later

## üí° PRO TIPS

1. **Screenshot EVERYTHING** - confirmation pages, reference numbers
2. **Call in the morning** - shorter hold times
3. **Be polite but firm** - "I need this resolved for a pending loan"
4. **Get names** - "May I have your name for my records?"
5. **Use "expedite"** - Ask if there's an expedite process
